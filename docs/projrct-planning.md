# Amass - Project Planning

## Problem Statement
University students often receive a limited monthly allowance, scholarship, or part-time income, making it important to manage their money carefully. However, many students do not record their daily expenses, especially small purchases such as food, transport, or mobile data. As a result, they often lose track of where their money goes, make impulse purchases, overspend, and struggle to build consistent saving habits. Existing methods such as notebooks and spreadsheets require manual effort and do not provide clear insights into spending patterns.

Amass is a web-based personal finance tracker designed primarily for university students. It enables users to record income and expenses quickly, monitor their available balance, visualize spending through interactive charts, set budgets and savings goals, and gain simple insights that encourage better financial decisions and healthier spending habits.


## Target Users
**Primary Users**
- University students (main focus for MVP)
- Young professionals
- Freelancers

**Secondary Users**
- Anyone who wants to manage personal finances and build better saving habits

## Competitor Analysis
**Applications Reviewed:** Money Manager Expense & Budget, Fast Budget, Spending Tracker

**Features I Like:**
- Double-entry style tracking (expenses/income auto-update account balance)
- Multiple account types tracked in one place, filterable
- Category-based budgets with visual spend-vs-limit comparison
- Calendar view for reviewing transactions
- No mandatory bank linking (privacy-friendly manual entry)

**Problems I Found:**
- Plain list views instead of clear at-a-glance summaries
- Unnecessary sub-category friction for simple purchases
- Ads in free versions
- Advanced features locked behind paywalls
- Longer-than-expected setup for new users

**Key Insight:** Amass should prioritize fast, low-friction transaction logging over advanced/rare features.

## Feature List
**Must Have (MVP)**
- Register / Login
- Add income
- Add expenses (fast entry, no forced sub-categories)
- Edit / Delete transactions
- Dashboard (clear summary view)
- Charts (spending by category)
- Budget planning (monthly limits per category)

**Should Have**
- Savings goals
- Monthly reports

**Nice to Have**
- Financial health score
- Spending predictions
- Smart insights
- Notifications

## User Stories
- As a user, I want to create an account so that I can securely manage my finances.
- As a user, I want to log in and log out so that my financial data stays private and protected.
- As a user, I want to record my income so that I know my available balance.
- As a user, I want to record my expenses quickly so that I can track my spending without friction.
- As a user, I want to edit or delete a transaction so that I can correct mistakes.
- As a user, I want to see a dashboard summary so that I instantly know if I'm overspending.
- As a user, I want to view charts of my spending by category so that I understand my habits.
- As a user, I want to set a monthly budget per category so that I can control my spending before it happens.
- As a user, I want to see how much of my budget I've used so that I can adjust my spending in real time.
